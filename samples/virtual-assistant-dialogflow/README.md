# Riva and Google Dialogflow Integration Demo User Guide

## Overview

This sample code demonstrates the integration of Google Dialogflow and Riva Speech Service in the form of Weather Chatbot Web Application.  
Google Dialogflow provides NLP and DM, whereas Riva provides ASR and TTS.  
Follow [this link](https://docs.nvidia.com/deeplearning/riva/user-guide/docs/samples/dialogflow.html) to learn more about the Riva and Dialogflow Integration Sample
